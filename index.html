Tutoriel WordPress pour mcroom.fr

1. Changer l'image de fond d'écran sur WordPress

1. Accéder au tableau de bord WordPress :

Connecte-toi au tableau de bord WordPress en allant sur https://mcroom.fr/wp-admin avec les identifiants d'administrateur.

2. Aller dans l'apparence du thème Aravalli :

Dans la barre de navigation à gauche, clique sur Apparence > Personnaliser. Le thème Aravalli propose des options spécifiques pour la personnalisation des couleurs et des arrière-plans.

3. Modifier l'image de fond :

Une fois dans l'outil de personnalisation du thème, clique sur Couleurs et arrière-plan ou Image d'arrière-plan. Téléverse une nouvelle image de fond ou choisis une image existante dans la bibliothèque de médias.

4. Ajuster les paramètres de l'image de fond :

Ajuste la position de l'image, sa répétition et sa taille. Pour le thème Aravalli, tu peux choisir une mise en page épurée et moderne. N'oublie pas de cliquer sur Publier pour enregistrer les changements.

2. Configurer WooCommerce

1. Lancer l'assistant de configuration WooCommerce :

Si ce n'est pas encore fait, après l'installation de WooCommerce, un assistant de configuration s'affiche automatiquement. Sinon, accède à WooCommerce > Statut > Outils > Relancer l'assistant de configuration.

2. Ajouter les informations de la boutique :

Adresse : Saisis l'adresse du magasin.
Devise : Choisis la devise utilisée pour les transactions.
Types de produits : Indique si tu vends des produits physiques, numériques ou des services.

3. Configurer les paiements :

Accède à WooCommerce > Réglages > Paiements. Active PayPal et configure-le en suivant les instructions d'intégration dans la section Configurer PayPal.

4. Configurer les expéditions (si besoin) :

Accède à WooCommerce > Réglages > Expédition. Définis les zones de livraison et les coûts associés.

5. Finaliser les réglages de base :

Passe en revue les autres options dans WooCommerce > Réglages, comme les taxes, les comptes clients, etc., pour ajuster selon les besoins du client.

3. Configurer WooPayments

1. Accéder à WooPayments :

Après avoir installé le plugin WooPayments, va dans WooCommerce > WooPayments.

2. Connexion à Stripe (ou autre passerelle) :

WooPayments utilise Stripe pour les paiements. Suis les instructions pour connecter Stripe en fournissant les informations bancaires.

3. Test des paiements :

Active le mode test pour vérifier si tout fonctionne correctement avant de passer en mode production.

4. Personnaliser les paramètres de paiement :

WooPayments permet de configurer les types de cartes acceptées et d'ajouter d'autres méthodes comme Apple Pay ou Google Pay.

5. Intégration de PayPal :

Une fois Stripe configuré, tu peux également intégrer PayPal. Un message dans l'administration WooCommerce te rappellera de connecter PayPal en allant dans WooCommerce > Paiements et en cliquant sur Activer PayPal.

4. Connecter votre boutique à Omnisend

1. Se connecter à Omnisend :

Omnisend est un outil marketing qui te permet d'automatiser tes campagnes emails et SMS. Pour connecter ta boutique, installe le plugin Omnisend.

2. Collecter des abonnés :

Utilise les formulaires d'inscription pour collecter des emails de tes visiteurs et les convertir en clients.

3. Automatiser les emails :

Configure des emails automatiques pour les paniers abandonnés ou les rappels de paiement.

Conseils supplémentaires :

Backups réguliers : Sauvegarde régulièrement le site avant toute modification importante.

Test des modifications : Teste les changements sur un site de staging ou utilise un mode maintenance.
