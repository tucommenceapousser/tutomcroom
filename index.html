Tutoriel WordPress pour mcroom.fr

1. Changer l'image de fond d'écran sur WordPress

1. Accéder au tableau de bord WordPress :

Connecte-toi au tableau de bord WordPress en allant sur https://mcroom.fr/wp-admin avec les identifiants d'administrateur.

2. Aller dans l'apparence du thème :

Dans la barre de navigation à gauche, clique sur Apparence > Personnaliser.

3. Modifier l'image de fond :

Une fois dans l'outil de personnalisation du thème, clique sur Couleurs et arrière-plan ou Image d'arrière-plan (selon le thème utilisé). Là, tu pourras téléverser une nouvelle image de fond ou choisir une image existante dans la bibliothèque de médias.

4. Ajuster les paramètres de l'image de fond :

Une fois l'image chargée, tu pourras ajuster sa position (centrée, à gauche, etc.), sa répétition, et sa taille (remplir l'écran, etc.). N'oublie pas de cliquer sur Publier pour enregistrer les changements.

2. Configurer WooCommerce

1. Lancer l'assistant de configuration WooCommerce :

Si ce n'est pas encore fait, après l'installation du plugin WooCommerce, un assistant de configuration s'affiche automatiquement. Sinon, accède à WooCommerce > Statut > Outils > Relancer l'assistant de configuration.

2. Ajouter les informations de la boutique :

Adresse : Saisis l'adresse du magasin.
Devise : Choisis la devise utilisée pour les transactions.
Types de produits : Indique si tu vends des produits physiques, numériques ou des services.

3. Configurer les paiements :

Accède à WooCommerce > Réglages > Paiements. Choisis les méthodes de paiement à activer (par exemple, Carte de crédit, PayPal, ou Stripe). Configure chaque méthode en ajoutant les informations de l'API si nécessaire (par exemple, en connectant Stripe pour les paiements par carte bancaire).

4. Configurer les expéditions (si besoin) :

Accède à WooCommerce > Réglages > Expédition. Définis les zones de livraison et les coûts associés.

5. Finaliser les réglages de base :

Passe en revue les autres options dans WooCommerce > Réglages comme les taxes, les comptes clients, etc., pour les ajuster selon les besoins du client.

3. Configurer WooPayments

1. Accéder à WooPayments :

Après avoir installé le plugin WooPayments, va dans WooCommerce > WooPayments.

2. Connexion à Stripe (ou autre passerelle) :

WooPayments se base souvent sur Stripe. Suis les instructions pour connecter Stripe en fournissant les informations de l'entreprise et les détails bancaires.

3. Test des paiements :

WooPayments propose souvent un mode test. Active-le pour vérifier si tout fonctionne correctement avant de passer en mode production.

4. Personnaliser les paramètres de paiement :

Tu peux configurer les types de cartes acceptées, ajouter d'autres méthodes de paiement comme Apple Pay, Google Pay, etc., en fonction des besoins du client.

Conseils supplémentaires :

Backups réguliers : Il est recommandé de sauvegarder régulièrement le site avant de faire des modifications importantes.

Test des modifications : Toujours tester sur un site de staging ou utiliser un mode maintenance pendant les ajustements.
